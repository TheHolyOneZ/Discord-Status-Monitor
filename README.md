# Discord Status Monitor

The **Discord Status Monitor** is a bot extension that allows you to display the status of your Discord bots, websites, and other services directly in a Discord channel. It's great for keeping your community informed about the health of your services.

## âœ¨ Features

* **Customizable Status Embed:** The bot creates a single, organized message that automatically updates with the status of your monitored services.

* **Monitor Anything:** Add and track the status of your own Discord bots, any website URL, and custom services (like your game server or database).

* **Official Discord Status:** Keep an eye on the official Discord services like the Gateway and Voice servers.

* **Easy Setup:** All management is handled through a simple in-app admin panel, so you don't need to edit any code.

* **Automatic Updates:** The status embed refreshes automatically at a set interval. You can also trigger a manual refresh at any time.

## ðŸ¤– Commands

The main command to get started is `/status-setup`.

### `/status-setup`

This command opens the main **Admin Panel** for the status monitor. From here, you can manage everything using a series of buttons:

* **Manage Bots:** Add or remove the bots you want to monitor.

* **Manage Websites:** Add or remove website URLs to track their status.

* **Manage Services:** Add or remove custom services and manually set their status.

* **Discord Services:** Choose which official Discord services (like API, Gateway, etc.) you want to display.

* **Post/Move Status:** Select a channel to post the status embed in. If you've already posted it, this button lets you move it to a new channel.

* **Set Interval:** Change how often the status embed refreshes automatically.

* **Change Title:** Customize the title of the status embed.

* **Refresh:** Manually update the status embed right away.
